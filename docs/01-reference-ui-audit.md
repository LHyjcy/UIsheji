# Reference UI Audit

## Visual Reading

The reference is an experimental designer portfolio landing page. The visual language is high-contrast black and white, large cinematic 3D character imagery, and offset image collages.

## Global Rules

- UI colors stay black, white, and gray.
- Saturated color comes from images and the six step highlight bars.
- Layout should not become a generic centered landing page.
- Cards use small soft radius on media, large radius on step cards, and pill radius only on controls.
- Large desktop widths preserve the reference composition; mobile can reflow but cannot overlap text or crop controls.

## Page Sections

1. Dark hero based on reference image 2:
   - Black cinematic stage.
   - Large red-haired character close-up.
   - Chinese tutorial title, `B7`, and small English metadata.
   - Three supporting image cards.

2. White gallery collage based on reference image 2:
   - Large white field.
   - 16 mixed-size cards positioned around a center pill button.
   - Corner labels: `LQB_2026`, `DESIGNER`, `EXCLUSIVE CONTENT`, `COLLECTION`.

## Wireframe Notes

Reference image 1 confirms the structure: hard section cuts, flat black/gray blocks, a central creative AI card motif, and asymmetric placeholder clusters. The final implementation uses the same spirit but now keeps only the hero and upper gallery content.
